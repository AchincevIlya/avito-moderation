# Avito Moderation Dashboard

Панель модерации объявлений с фильтрами, сортировками, карточками и полной страницей объявления.

## Основной функционал

###  Фильтрация и поиск
- Поиск по названию
- Фильтр по статусу
- Фильтр по категории
- Фильтр по цене
- Сортировка по дате, цене, приоритету

###  Пагинация
- Серверная пагинация
- Удобная навигация

###  Карточки объявлений
- Фото
- Цена, категория, статус
- "Срочно" / "Обычное"

###  Страница объявления
- Галерея изображений
- Полное описание
- Характеристики
- Продавец
- История модерации
- Статистика действия

###  Модерация
- Одобрить
- Отклонить (с причиной)
- На доработку (с причиной)
- Оптимистические обновления

###  Горячие клавиши

 - "A" - Одобрить 
 - "D" - Отклонить 
 - "R" - Запросить изменения 
 - "←"  Предыдущее 
 - "→"  Следующее 

## Технологии
- React 18  
- TypeScript  
- React Query  
- React Router  
- Material UI  
- Axios  

##  Установка и запуск

### 1. Запуск сервера

cd tech-int3-server
npm install
npm run dev

### 2. Переход в папку с фронтом

cd avito-moderation

### 3. Установка зависимостей

npm install

### 4. Запуск локального сервера

npm run dev
